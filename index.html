Gestion des Équipements Informatiques

Système de gestion des équipements informatiques permettant le suivi des pannes, des tickets de support, et la génération de rapports et statistiques.

En savoir plus sur la Gestion des Équipements Informatiques
Description du projet :

Le projet de gestion des équipements informatiques permet à une organisation de suivre l’état des équipements, de gérer les pannes, et d'assurer le suivi des tickets de support. Ce système permet également de générer des rapports détaillés et des statistiques pour une gestion optimale des ressources et des incidents. L'objectif est de garantir une prise en charge rapide des problèmes et de maintenir un suivi de la performance de l'infrastructure informatique.

Technologies utilisées :
  • Backend : Spring Boot, Spring Data JPA, Spring Security
  • Frontend : Angular 16
  • Base de Données : MySQL
  • Test Unitaire : JUnit
  • Conteneurisation : Docker
Fonctionnalités :
  • Suivi et gestion des équipements informatiques
  • Gestion des pannes et tickets de support
  • Génération de rapports et de statistiques sur les équipements
  • Tableau de bord pour l'administration et la gestion des tickets
  • Notifications et rappels pour les maintenances et réparations
Voir le repository GitHub
ConstructionXpert Services

Application de gestion de projets de construction. Permet de planifier, organiser et suivre les projets de construction.

En savoir plus sur ConstructionXpert Services
Description du projet :

L'application ConstructionXpert Services est un outil de gestion de projets de construction. Elle permet à l'équipe de gestion de projets de suivre et d'organiser les différentes étapes d'un projet de construction, de la planification à la réalisation, en passant par le suivi des ressources et des tâches.

Technologies utilisées :
  • Frontend : HTML, CSS, Bootstrap, Tailwind, JavaScript
  • Backend : Java EE (Servlets, JSP)
  • Base de Données : MySQL
  • Communication avec la base de données : JDBC API
  • Serveur d'Application : Apache Tomcat
  • Outils de Développement : IntelliJ IDEA, Maven
  • Système de Gestion de Versions : Git
Fonctionnalités :
  • Gestion des projets de construction
  • Suivi des tâches et des ressources
  • Interface conviviale et facile à utiliser
  • Gestion des utilisateurs et des rôles
  • Système de notifications et rappels pour les deadlines
